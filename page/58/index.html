What is the Sigstore project?

2022-02-22 10:09:35
tutorials
1177 words 6 mins read
Sigstore is the latest project of Linux Foundation, which is a new model of software signature based on Certificate Transparency Log. Sigstore solves the problem of keeping private keys caused by traditional PGP signatures, it has excellent openness and scalability, and is known as the Let’s Encrypt of software signatures. TL;DR Briefly, what the Sigstore project implements is to record the information of each signature through the Certificate Transparency Log, and end users query the Log and verify it when they need to use it.
